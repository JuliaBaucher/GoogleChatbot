System Architecture

This page is a static, single-file web application. When the user sends a message, the browser makes a POST /chat request to an AWS API Gateway HTTP API endpoint. The request is handled by an AWS Lambda function, which calls a large language model and returns a JSON response containing the assistant’s reply.
